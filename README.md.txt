


  Pedir al usuario su numero de tarjeta.
  
  Crea una función llamada isValidCard.
	El string ingresado sera convertido en array e invertido el orden de sus elementos.
	 Recorreremos el string desde la segunda posicion e iremos de 2 en 2, para obtener las posiciones par.
	 Multiplicaremos esos elementos por 2.
		Si son mayores que 9 se les restara 9 para obtener numeros de un solo digito.
	Luego sumaremos cada elemento para obtener un solo numero.
		Dividiremos ese numero por 10 y si el resto es 0, la tarjeta es válida.
